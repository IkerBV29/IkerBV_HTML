CREATE TABLE examples

  • Basic CREATE TABLE statement
  • Creating a table with a particular storage engine
  • Creating a table with auto_increment
  • Creating a table with the current timestamp
  • Creating a table with TIMESTAMP DEFAULT NOW()
  • Viewing a table definition

Basic CREATE TABLE statement

CREATE TABLE t (
    i INT,
    c CHAR(10)
);

Creating a table with a particular storage engine

CREATE TABLE t (
    i INT
) ENGINE = InnoDB;

To create a table explicitly in a given storage engine. The default storage engine is used unless you specify a different one.

Creating a table with auto_increment

CREATE TABLE t (
    i INT NOT NULL AUTO_INCREMENT,
    PRIMARY KEY (i)
);

Creating a table with the current timestamp

CREATE TABLE t (
    ts TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

Creating a table with TIMESTAMP DEFAULT NOW()

CREATE TABLE t (
    ts TIMESTAMP DEFAULT NOW()
);

Since MySQL 5.6.5, you can use CURRENT_TIMESTAMP as the default value for a column. Alternatively, use the NOW() function.

Viewing a table definition

SHOW CREATE TABLE t;

The SHOW CREATE TABLE statement returns the SQL statement that creates the specified table.
